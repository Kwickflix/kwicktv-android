## Features

- **Live TV** with a Plex-style timeline **TV Guide** (now line, genre tabs, logos)
- **Movies & Series** with posters, ratings, cast, and **Resume / Play from Beginning**
- **Auto-play next episode**, favorites, continue watching, and multiple profiles
- **Picture-in-Picture**, sleep timer, screen lock, and a built-in proxy
- **Parental controls** with a PIN
- **Multiple providers** — Xtream Codes and M3U, add as many as you like
- **Hide & reorder** categories and channels to taste
- **Themes**, including retro **Fallout** (Pip-Boy terminal) and **Final Fantasy** looks
- Works with a **remote** (Android TV / Fire TV) or **touch** (phone / tablet)
